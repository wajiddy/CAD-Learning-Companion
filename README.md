# CAD Learning Companion (Android) (Gemvision-Aligned)

Interactive 3D shape viewer built in Android Studio using OpenGL ES 2.0.

## Features
- Choose a 3D shape: Triangle (pyramid), Square (cube), Rectangle (rectangular prism)
- Drag to rotate the 3D model
- Toggle vertex points ON/OFF
- Edit Mode: select a vertex and drag to adjust its position (basic X/Y editing)

## How to Run
1. Clone the repo
2. Open in Android Studio
3. Run on an emulator or Android device

## Tech
- Java
- Android Studio
- OpenGL ES 2.0
